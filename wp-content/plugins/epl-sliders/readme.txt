Convert the featured image for each listing into a dynamic slider.

== Description ==

Convert the featured image for each listing into a dynamic slider with customisable display options.

== Installation ==

1. Install the plugin from Dashboard > Plugins > Add New > Upload. Select the downloaded plugin and activate.
